# Flipkart Listing Guidelines (Fashion, Apparel, Footwear, Accessories)

This document describes the listing-specific SEO, content, and structure rules for Flipkart fashion categories.

---

# 1. General Principles

Flipkart emphasizes:

- Accuracy  
- Relevance  
- Attribute completeness  
- Non-promotional content  
- Clear differentiation between product types  

Search ranking is influenced by:

- Title optimization  
- Attribute coverage  
- Conversion rate  
- Listing Quality Score (LQS)

---

# 2. Title Guidelines

A good Flipkart title is:

- 60–120 characters  
- Compact  
- Includes 2–3 important keywords  
- Accurate and non-promotional  

## 2.1 Recommended Structure

Brand + Product Type + Key Feature + Pattern / Material + Color


### Example
- “BrandA Men’s Running Shoes Mesh Lightweight Blue”
- “BrandB Women’s Cotton Kurta Floral Print Pink”

## 2.2 Allowed
- Key keywords (running shoes, kurta, wallet)
- Fabric/material (cotton blend, mesh)
- Pattern (solid, printed)
- Fit or type (slim fit, regular fit)

## 2.3 Not Allowed
- Promotional keywords:
  - “Best Price”
  - “Hot Deal”
  - “Sale”
- ALL CAPS or mixed caps
- Competitor brand names
- Repetitive keyword stuffing
- Irrelevant traffic-bait terms

---

# 3. Bullet Points (Key Features)

Flipkart recommends **4–6 bullets**.

Bullets should:

- Highlight important attributes  
- Remain factual and concise  
- Use simple grammar  
- Match images  

## 3.1 Good Bullet Structure
- **FABRIC:** Cotton blend with soft finish  
- **FIT TYPE:** Regular fit for daily wear  
- **PATTERN:** Floral print  
- **CLOSURE:** Lace-up (for shoes)  
- **CARE INSTRUCTIONS:** Machine wash  

## 3.2 Don’ts
- No medical claims  
- No performance guarantees  
- No unverifiable quality promises  
- No exaggerated adjectives (“premium”, “superior”) without evidence  
- No irrelevant lifestyle claims  

---

# 4. Product Description

Purpose: improve conversions + answer customer doubts.

Should include:

- 2–3 small paragraphs  
- Material details  
- Use-case suggestions  
- Fit or design notes  
- Feature elaboration (breathability, cushioning)  
- Manufacturing details (if relevant)  

Avoid:

- Long blocks of text  
- Keyword stuffing  
- Promotional statements  
- Repeating bullets verbatim  

---

# 5. Attribute Requirements (Fashion)

Flipkart Fashion SEO depends heavily on **attribute completeness**.

Below are mandatory/critical attributes per category.

## 5.1 Apparel (Shirts, Kurtas, Dresses)
- Fabric  
- Fit type  
- Pattern  
- Sleeve type  
- Neck type  
- Occasion  
- Color  
- Size chart  

## 5.2 Sarees / Ethnic Wear
- Fabric type (silk, chiffon, poly-silk)  
- Border style (zari, woven)  
- Pattern  
- Blouse inclusion  
- Length & weight  
- Wash care  

## 5.3 Footwear
- Upper material (mesh, PU, textile)  
- Sole material (EVA, rubber, TPR)  
- Closure (lace-up, slip-on)  
- Pattern  
- Color  
- Insole features  

## 5.4 Accessories
- Material (PU, canvas, synthetic)  
- Compartments  
- Closure  
- Dimensions  
- Color  

---

# 6. Prohibited Listing Content

Flipkart will suppress listings containing:

- False material claims (“100% leather”, “pure silk”)  
- Medical claims  
- Performance guarantees (“makes running faster”)  
- Competitor names  
- Irrelevant keywords  
- Promotional text in title/description  
- Misleading pattern claims (printed vs woven mismatch)  

---

# 7. What the Copilot Should Do With This

When optimizing Flipkart listings:

1. Generate title with correct structure  
2. Ensure attribute completeness  
3. Suggest 4–6 factual bullets  
4. Avoid prohibited words  
5. Check for consistency between images + bullets + attributes  
6. Suggest long-tail backend keywords (in description or metadata)
