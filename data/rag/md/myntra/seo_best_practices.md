# Myntra SEO & Listing Quality Best Practices  
Applicable to Apparel, Footwear, Accessories

Myntra’s ranking algorithm is heavily tied to **Listing Quality Score (LQS)**.  
SEO revolves around fashion accuracy, attribute completeness, and visual quality.

---

# 1. Core SEO Principles

1. Brand-first structure in titles  
2. Fashion-relevant keywords  
3. High attribute completeness  
4. Consistency across title → bullets → images → attributes  
5. Accurate materials & patterns  
6. Detailed product attributes  
7. High-resolution images  

Unlike other marketplaces, Myntra directly penalizes:

- attribute gaps  
- misleading materials  
- unrealistic colors  
- high return rates  

---

# 2. Title SEO Guidelines

Best format:

Brand + Product Type + Material + Pattern + Color


Examples:
- “BrandA Women Solid Rayon Straight Kurta Pink”
- “BrandB Men Running Shoes Mesh Navy Blue”

Include:
- fabric  
- pattern  
- color  
- gender  

Avoid:
- repeated terms  
- generic adjectives  
- promotional tone  

---

# 3. Bullet SEO Guidelines

Add searchable descriptors:
- sleeve length  
- neck type  
- fit  
- pattern name  
- hemline  
- style descriptors (A-line, tapered)  

Good bullets improve conversion and LQS.

---

# 4. Attribute SEO (Most Important Factor)

Must-fill fields:

## Apparel
- Fabric  
- Pattern  
- Fit  
- Length  
- Sleeve  
- Neck  
- Weave type  
- Transparency  
- Occasion  
- Hemline  

## Footwear
- Upper material  
- Sole material  
- Closure  
- Pattern  
- Toe shape  
- Insole type  
- Technology  

Complete attributes → Higher ranking.

---

# 5. Keyword Strategy

Use:
- “slim fit shirt”  
- “cotton blend kurta”  
- “mesh running shoes navy”  
- “printed A-line dress”  

Avoid:
- irrelevant trending terms  
- keyword stuffing  
- mismatched material tags  

---

# 6. Image SEO Rules

Listings with:
- 1500×1500 images  
- model shots  
- 4+ views  

rank significantly higher.

Avoid:
- low-quality lighting  
- inconsistent colors  
- edited skin tones  
- props not included  

---

The copilot should:

1. Rewrite titles to Myntra’s brand-first SEO style  
2. Suggest premium, fashion-accurate bullets  
3. Ensure full attribute completion  
4. Add high-value keywords
