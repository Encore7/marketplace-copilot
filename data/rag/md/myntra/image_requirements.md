# Myntra Image Requirements  
Applicable to Apparel, Footwear, Accessories

Myntra follows premium fashion photography standards.  
Images directly influence LQS (Listing Quality Score), CTR, and return rate.

---

# 1. General Image Standards

- Minimum resolution: **1500×1500 px**  
- High-quality, well-lit photography  
- Accurate color representation (no heavy editing)  
- No watermarks or text overlays  
- PNG/JPEG format only  
- White or light neutral backgrounds for studio shots  

---

# 2. Required Images Per Category

## 2.1 Apparel
Minimum 4 images:
- Front  
- Back  
- Close-up of fabric  
- Model wearing the product (mandatory for top categories)  

## 2.2 Footwear
Minimum 5 images:
- Side  
- Front  
- Top  
- Back  
- Sole  
- Material close-up  

## 2.3 Accessories (bags, wallets)
- Front  
- Back  
- Inside compartments  
- Size reference  
- Material close-up  

---

# 3. Model Photography Guidelines (Strict)

Allowed:
- Natural poses  
- Clean background  
- Fashion-standard angles  

Not allowed:
- Suggestive poses  
- Heavy filters  
- Celebrity-look images  
- Distracting backgrounds  
- Over-editing skin tones  

Model shots must reflect **actual fabric drape** and silhouette.

---

# 4. Technical & Formatting Requirements

- Ratio: 1:1 recommended  
- No borders  
- No composite collages  
- File size < 3MB  
- True-to-life color accuracy  

---

# 5. Image Content Restrictions

Prohibited images:
- watermarks  
- logos (unless genuine brand)  
- unrealistic color saturation  
- stock photos  
- showing accessories not included  
- misleading props  
- lists of benefits or features inside the image  

---

# 6. Advanced Fashion Guidelines

- Fabric texture must be accurately visible  
- Stitching details must be clear  
- Embellishments must match the product  
- Footwear sole cushioning must be visible if claimed  

---

# 7. How The Copilot Should Use This File

The copilot should:

1. Check if required image views are present  
2. Flag over-edited or misleading images  
3. Recommend additional studio shots  
4. Detect mismatched colors  
5. Enforce Myntra’s high-end visual standards  
6. Suggest model-shot improvements
